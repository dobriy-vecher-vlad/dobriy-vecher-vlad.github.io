Чтобы открыть учебник, необходимо открыть файл index.html в браузере.

Папка FontAwesome содержит в себе различные иконки для сайта.
Папка pages содержит в себе данные страниц.
Файл functions.js содержит в себе весь функционал сайта.
Файл main.css содержит в себе все стили сайта.
Файл values.js содержит в себе оглавление и ссылки на страницы.
